# Filesystem Documentation
### Raw Data
We will use the original dataset name as provided by the source. Keep all original file extensions.

### Cleaned Data
We will use desriptive names that reflect content and processing stage. For the Cleaned data we will include "clean" in the dataset. Then while integrating include descriptors such as "merged" and "final"

### Analysis and Results
We will name files according to content or purpose. Using underscores.

### Scripts
Use lowercase filenames with underscores. Include a descriptive verb for the purpose of the script

### Documentation
Will have the same name as the script adding "_documentation" to the end.

# Script [data_analysis](../scripts/data_analysis.py)

This script performs statistical analysis and visualization on the integrated NBA dataset, final_df, generated by the data integration process
The primary objective is to evaluate relationships between player performance and salary. 

This script uses: pandas, matplotlib, seaborn, and statsmodels.

1. Load the integrated dataset
2. Define independent variables of PTS, AST, REB, TOV, GP
3. Fit an Ordinary Least Swaures model
4. Compute corrleations for Salary, PTS, ASt, REB, TOV, and GP
5. Identify the most predictive performance metric for salary
6. Generate a scatter plot of Salary vs PTS
